<p>약수의 개수가 세 개 이상인 수를 합성수라고 합니다. 자연수 <code>n</code>이 매개변수로 주어질 때 <code>n</code>이하의 합성수의 개수를 return하도록 solution 함수를 완성해주세요.</p>

<ul>

</ul>

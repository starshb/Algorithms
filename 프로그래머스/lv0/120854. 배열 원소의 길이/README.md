### 문제 설명

<p>문자열 배열 <code>strlist</code>가 매개변수로 주어집니다. <code>strlist</code> 각 원소의 길이를 담은 배열을 retrun하도록 solution 함수를 완성해주세요.</p>

<hr>

<h5>제한사항</h5>

<ul>
<li>1 ≤ <code>strlist</code> 원소의 길이 ≤ 100</li>
<li><code>strlist</code>는 알파벳 소문자, 대문자, 특수문자로 구성되어 있습니다.</li>
</ul>

<hr>

</ul>

</ul>

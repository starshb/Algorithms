### 문제 설명

<p>정수 배열 <code>array</code>가 매개변수로 주어질 때, 가장 큰 수와 그 수의 인덱스를 담은 배열을 return 하도록 solution 함수를 완성해보세요.</p>

<hr>

<h5>제한사항</h5>

<ul>
<li>1 ≤ <code>array의</code> 길이 ≤ 100</li>
<li>0 ≤ <code>array</code> 원소 ≤ 1,000</li>
<li><code>array</code>에 중복된 숫자는 없습니다.</li>
</ul>

</ul>

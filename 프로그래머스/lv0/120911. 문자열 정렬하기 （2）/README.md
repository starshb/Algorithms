### 문제 설명

<p>영어 대소문자로 이루어진 문자열 <code>my_string</code>이 매개변수로 주어질 때, <code>my_string</code>을 모두 소문자로 바꾸고 알파벳 순서대로 정렬한 문자열을 return 하도록 solution 함수를 완성해보세요.</p>

<hr>

<h5>제한사항</h5>

<ul>
<li>0 &lt; <code>my_string</code> 길이 &lt; 100</li>
</ul>

</ul>

</ul>

</ul>

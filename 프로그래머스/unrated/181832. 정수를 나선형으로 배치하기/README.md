### 문제 설명

<p>양의 정수 <code>n</code>이 매개변수로 주어집니다. <code>n</code> × <code>n</code> 배열에 1부터 <code>n</code><sup>2</sup> 까지 정수를 인덱스 [0][0]부터 시계방향 나선형으로 배치한 이차원 배열을 return 하는 solution 함수를 작성해 주세요.</p>

<hr>

<h5>제한사항</h5>

<ul>
<li>1 ≤ <code>n</code> ≤ 30</li>
</ul>

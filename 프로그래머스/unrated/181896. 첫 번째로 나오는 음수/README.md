### 문제 설명

<p>정수 리스트 <code>num_list</code>가 주어질 때, 첫 번째로 나오는 음수의 인덱스를 return하도록 solution 함수를 완성해주세요. 음수가 없다면 -1을 return합니다.</p>

<hr>

<ul>
<li>5 ≤ <code>num_list</code>의 길이 ≤ 100</li>
<li>-10 ≤ <code>num_list</code>의 원소 ≤ 100</li>
</ul>
